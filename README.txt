## Overview
GovSec AI is an AI-powered system that detects phishing and fake websites impersonating Kenyan government services.
It enhances cybersecurity for e-Government platforms and protects citizen data from malicious actors.

## Features
- AI-based phishing detection (Random Forest)
- Real-time URL analysis via Streamlit dashboard
- Threat distribution visualization
- Expandable for predictive analytics & API alerts

## Tech Stack
Python, scikit-learn, Streamlit, Plotly, BeautifulSoup, Requests

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Train the model: `python train_model.py`
3. Launch the dashboard: `streamlit run app.py`
